Notice

Le shortcode Notice permet d’afficher 4 types de message pour vous aider à structurer votre contenu.

Note

{{% notice note %}}
Une notice de type *note*
{{% /notice %}}

s’affiche comme

Une notice de type note

Info

{{% notice info %}}
Une notice de type *info*
{{% /notice %}}

s’affiche comme

Une notice de type info

Tip

{{% notice tip %}}
Une notice de type *tip*
{{% /notice %}}

s’affiche comme

Une notice de type tip

Warning

{{% notice warning %}}
Une notice de type *warning*
{{% /notice %}}

s’affiche comme

Une notice de type warning
