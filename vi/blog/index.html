Thủ thật sử dụng Notepad++ trong công việc

in tools

4/1/2018

Giới thiệu Khi tôi làm việc, tôi code C#, HTML, Javascript…Và phần lớn thời gian tôi phải xử lý code, chủ yếu là string và text. Và tôi thường sử dụng Nodepad++ để giúp code nhanh hơn vì nó là công cụ mạnh và có nhiều chức năng rất hay. Sau đây tôi sẽ chia sẻ một vài chức năng mà tôi hay dùng mà có thể có bạn chưa biết, hi vọng sẽ giúp ích được cho các bạn trong công việc.

Site param

1/1/0001

Les shortcode siteparam est utilisé pour vous aider à afficher des valeurs provenant des paramètres globaux du site. Par exemple, dans ce site, le paramètre editURL est utilisé dans le fichier config.toml [params] editURL = "https://github.com/matcornic/hugo-theme-learn/edit/master/exampleSite/content/" Utilisez le shortcode siteparam pour affichier sa valeur.
